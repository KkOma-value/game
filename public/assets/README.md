# Rift Salvage Imagegen Assets

The code expects these project-bound bitmap files:

- `player-drone.png`
- `enemy-chaser.png`
- `enemy-drifter.png`
- `enemy-orbiter.png`
- `boss-machine.png`
- `energy-core.png`
- `hazard-mine.png`
- `rift-background.png`
- `title-badge.png`
- `warning-icon.png`
- `goal-token.png`

Gameplay sprites and UI flavor assets should be transparent PNGs generated via Codex imagegen on a flat chroma-key background and then keyed locally. The rift background should be a seamless tileable bitmap.
